== Rainforest

Description: 

  This is a smaller practice version of  the amazon.com online store.
  
Learning objective:

  In the first iterartion of this project the learning objective was to practice building apps in the ruby on rails framework, and to better  understand the concept of MVC in rails. In the second interation, the objective was to apply AJAX to the project and to gain a better understanding of how the AJAX tool works. AJAX was implemented for infitie scrolling and also on the forms.This is a completed project.
